# 软通新闻智能整理器

## 项目简介

48小时限时开发Web应用，实现软通30天新闻智能整理，支持新闻解析、摘要提取、关键词分析、数据统计、检索筛选功能。

## 技术栈

- 前端：原生JS + TailwindCSS
- 部署：Gitee Pages（国内免费静态托管）
  
  ## 核心功能
1. 支持批量粘贴20+篇新闻内容，自动解析格式
2. 智能提取标题、发布日期、正文、关键词
3. 全局数据统计：总篇数、时间范围、高频关键词
4. 实时搜索筛选新闻内容
5. 内置20篇示例新闻，一键加载演示
   
   ## 使用方式
6. 在输入框粘贴新闻内容（多篇使用空行分隔）
7. 点击【AI智能整理】生成结果
8. 可使用搜索框筛选目标新闻
9. 点击【加载示例】快速查看演示效果
